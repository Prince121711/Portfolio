Place your permanent certificate files in this folder.

Examples:
- `certificates/ai-native-workshop.pdf`
- `certificates/mongodb-workshop.jpg`

After adding a file here, update the `staticAssets.certificates` array inside `portfolio.html`.

Example entry:

```js
{
  title: "AI Native Workshop",
  subtitle: "NIT Trichy participation certificate",
  url: "certificates/ai-native-workshop.pdf",
  isPdf: true,
  downloadName: "AI_Native_Workshop.pdf"
}
```
